Web Accessibility Guide (WAG)

WAG is a toolset that makes World Wide Web friendlier to visually impaired people. It is currently based on WCAG (Web Content Accessibility Guideline) and WAI-ARIA (Web Accessibility Initiative â€“ Accessible Rich Internet Applications) guidelines by W3, but is designed to easily extend beyond that in future. Three kinds of visual impairments are considered in WAG; total blindness, low vision and color blindness. On one side WAG assists developers and reviewers to identify accessibility flaws in web sites and applications while also enabling visually impaired people to apply accessibility corrections automatically for web sites they access.

Wag DevKit

WAG DevKit is designed for Web development teams.It is capable of Evaluating web pages based on W3 guidelines. The DevKit is Capable of evaluating an entire web site once and Evaluation results are grouped by page. The User interface resembles a typical Integrated Development Environment (IDE). The DevKit provides Violation description, solution suggestions and examples side by side.It is also capable of evaluating a web site hosted in a private IP.DevKit keeps the evaluation history and shows improvement over time using graphs. It is capable of organizing data in multiple projects and its also designed to be plugged in as a component in Continuous Integration.


Wag Report

WAG Report tool is designed for Authorities & Reviewers which evaluates public web sites and capable of evaluating an entire web site at once. Wag Report displays the distribution of violations using specialized graphs.It also Provides indicators on accessibility level of a web site.Wag report feature do filtration of rules based on visual impairment and priority level.And also highlights accessibility flaws according to criticality.

Wag Assistant

WAG assistant is a tool designed for Visually impaired people and Organizations that support visually impaired people WAG Assistant runs as an integrated component to the web browser. It is capable of automatically applying fixes to accessibility flaws. The cool thing about Wag Assistant is that it can be triggered by keystrokes and also it is capable of triggering the screen reader.

Wag Community

Wag Community is designed for Researchers and Developers who are willing to contribute to the WAG rulebase. Wag Community has a Web interface to submit new rules.It has the Capability to test the associated script before submission. Wag Community does the Rule review process and also displays contributor information.
